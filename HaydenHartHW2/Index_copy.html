Concepts

CSS Box Model

  1. Padding: Clears an area around the content
  2. Margin: Clears an area outside the border
  3. Border: Border that goes around the padding and the content
  4. Content: The content in the box where the text appears

Padding

Padding is clearing out the area around the content. This padding is 25px.

Padding is always right outside the content. This padding is 50px by 25px.

Padding is how much room the content has until the next content. This padding is 35px, 25px, 45px, 55px.

Margin

Margin clears an area outside the borders. This margin is left default.

Margin is set outside the border. The left margin is set to 50px.

Margin creates space around the element. The left margin is set to 120px.
